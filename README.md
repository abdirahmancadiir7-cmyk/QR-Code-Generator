# QR Code Generator

A simple QR Code Generator built with Python. This project allows users to generate QR codes from text, URLs, or any other input and save them as PNG images.

## Features

- Generate QR codes from text
- Generate QR codes from URLs
- Save QR code as PNG image
- Simple command-line interface
- Easy to customize

## Requirements

- Python 3.x

## Installation

Clone the repository:

```bash
git clone https://github.com/abdirahmancadiir7-cmyk/qr-code-generator.git
```

Move into the project directory:

```bash
cd qr-code-generator
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the program:

```bash
python3 generator.py
```

## Example

```text
Enter text or URL:
https://github.com

Enter file name:
github

QR Code saved as github.png
```
